このサイトの PRO モードは偽のターミナルです — ただし最良の意味での偽物:*表示するもの*はすべて本物で、シェル自体だけが演出です。生きているように感じさせるために何をしたかを紹介します。

## 錯覚のレイヤー

ターミナルエミュレータは大きな機能ではなく、小さなディテールで納得させます:

- **ブートシーケンス。** PRO モードに入ると、プロンプトが現れる前に systemd 風の `[  OK  ]` 行が 110ms のリズムで再生されます。純粋な演出、即座の説得力。
- **タイプライター出力。** コマンドの出力は一度に現れず、キューから1行あたり約26msで流れ出ます。`Ctrl+L` でクリア、`Ctrl+C` はお馴染みの `^C` を出力。
- **Tab 補完。** コマンド語彙のトライ木。曖昧なプレフィックスは bash のように候補を列で表示します。
- **`setTimeout` の連鎖ではなくキュー。** すべての出力は単一のドレインループを通るため、速いタイピストでも2つのコマンドの出力が混ざりません。

## リアルのレイヤー

説得力の源は、データがモックではないことです:

- `htop` は Server-Sent Events のストリームを開き、ページを配信している Raspberry Pi の実際の CPU・RAM・SoC 温度を、直近60秒の ASCII スパークライン付きで表示。
- `neofetch` は `/api/v1/sysinfo` に問い合わせ — 本物のカーネルバージョン、本物の稼働時間、本物のロードアベレージ。
- `weather` はサーバーが物理的に存在するコンスタンサのライブ天気を Open-Meteo 経由(10分キャッシュ)で表示。
- `guestbook sign <メッセージ>` は Pi 内の SQLite データベースに書き込みます(IP ごとに毎分1署名、上限500行)。

## 楽しさのレイヤー

開発者は他人のターミナルで必ず同じことを打ちます。全部対応済みです:

- `sudo rm -rf /` → `/var/lib/dreams` へのドラマチックな降下、「防衛本能」により中断。
- `sudo apt update` → もっともらしい Debian の更新が、頼んでいない動画を開いて終わります。
- `matrix` → canvas 上のフルスクリーンのカタカナの雨。どのキーでも目が覚めます。
- `cowsay`、`fortune`、そして `fortune | cowsay` も動きます。
- `typing` → WPM テスト。ターミナルは何よりキーボードの場所ですから。

## これから作る人に伝えたいこと

1. 出力を正直に。本物の数字1つは偽のダッシュボード10個に勝ります。
2. 筋肉の記憶を尊重する:履歴は矢印、補完は Tab、Ctrl+L、Ctrl+C。これを壊すと魔法が解けます。
3. モバイルは別の契約が必要 — 16px の入力(でないと iOS がズーム)、スクロール時のフォーカス奪取なし、タイピングの代わりにコマンドチップ。
